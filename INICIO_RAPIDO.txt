INICIO RAPIDO
=============

1. Abrir la app:
   python cli.py run

2. Generar dashboards desde terminal:
   python cli.py generate S186

3. Validar proyecto:
   python cli.py validate

4. Ver estado:
   python cli.py status

Notas importantes:

- Los CSV activos siguen en data/contratistas/.
- El historico de cortes valido vive en output/exports/.
- El archivo correcto de bloques liberados es output/exports/bloques_liberados.html.
- No uses ni regeneres artefactos bloques_liberados.* en el root del proyecto.
